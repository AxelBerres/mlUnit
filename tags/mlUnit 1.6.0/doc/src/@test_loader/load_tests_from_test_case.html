load_tests_from_test_case

PURPOSE

test_loader/load_tests_from_test_case returns a test_suite with all test* methods from a test_case.

SYNOPSIS

function suite = load_tests_from_test_case(self, test_case_class)

DESCRIPTION 

test_loader/load_tests_from_test_case returns a test_suite with all
test* methods from a test_case.
  It returns an empty matrix, if the test is not found.

  Example
  =======
         loader = test_loader;
         suite = test_suite(load_tests_from_test_case(loader, 'my_test'));

CROSS-REFERENCE INFORMATION

This function calls:
  • get_test_case_names test_loader/get_test_case_names returns a list of string with all test* methods from the test_case_class.
  • map test_loader/map returns a list of objects instantiated from the class test_case_class and the methods in test_names.
  • test_suite test_suite contructor.
This function is called by:
  • gui gui_test_runner/gui execute the graphical user interface of mlUnit.
  • run text_test_runner/run execute the test and writes the results to a stream in textual form (using TextTestResult).
