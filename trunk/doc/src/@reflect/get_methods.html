get_methods

PURPOSE

reflect/get_methods returns the list of methods of the 'reflected' class.

SYNOPSIS

function meths = get_methods(self)

DESCRIPTION 

reflect/get_methods returns the list of methods of the 'reflected' class.

  Example
  =======
         r = reflect('test_case');
         get_methods(r);           % Returns a cell array with all methods
                                   % of the class test_case
