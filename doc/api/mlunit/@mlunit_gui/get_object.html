get_object

PURPOSE

mlunit_gui/get_object returns the singleton object of the gui window.

SYNOPSIS

function self = get_object(self) %#ok

DESCRIPTION 

mlunit_gui/get_object returns the singleton object of the gui window.

  Example
  =======
         run(mlunit_gui, 'mlunit_all_tests');
         runner = get_object(mlunit_gui)

  See also mlunit_gui.
