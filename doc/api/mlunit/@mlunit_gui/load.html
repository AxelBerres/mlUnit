DESCRIPTION 

mlunit_gui/loads starts the graphical user interface of mlUnit with
saved parameters from the .mat-file mlunit.tmp.

  Example
  =======
         load(mlunit_gui);

  See also mlunit_gui, mlunit_gui/SAVE.
