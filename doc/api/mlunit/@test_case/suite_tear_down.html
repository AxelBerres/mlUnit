suite_tear_down

DESCRIPTION 

test_case/suite_tear_down is called once after every test of this suite has
executed. Perform final suite cleanup here.

  suite_tear_down is not called on any particular test_case object, therefore
  it cannot set or query any object states, i.e. member variables.
