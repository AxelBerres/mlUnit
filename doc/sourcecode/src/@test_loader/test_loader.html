test_loader constructor.

  Class Info / Example
  ====================
  The class test_loader is able to create a test_suite with all
  test* methods from a test_case. 
  Example:
         loader = test_loader;
         suite = test_suite(load_tests_from_test_case(loader, 'my_test'));
