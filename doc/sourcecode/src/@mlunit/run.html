run

PURPOSE

mlunit/run executes the the graphical user interface of mlUnit.

SYNOPSIS

function self = run(self, dock)

DESCRIPTION 

mlunit/run executes the the graphical user interface of mlUnit.

  EXAMPLE
  =======
  Run in window mode:
         run(mlunit);
  Run in docked mode:
         run(mlunit, 1);

  See also MLUNIT, GUI_TEST_RUNNER/RUN.
