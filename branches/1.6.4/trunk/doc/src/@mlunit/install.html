install

PURPOSE

mlunit/install installs mlUnit. The current version of this method only renames the assert method of mlUnit, if a built-in assert already exists (which is the case for MATLAB 6.6).

SYNOPSIS

function self = install(self)

DESCRIPTION 

mlunit/install installs mlUnit. The current version of this method only
renames the assert method of mlUnit, if a built-in assert already exists
(which is the case for MATLAB 6.6).

  EXAMPLE
  =======
         install(mlunit);
