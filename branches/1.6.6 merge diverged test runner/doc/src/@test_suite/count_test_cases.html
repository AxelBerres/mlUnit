count_test_cases

PURPOSE

test_suite/count_test_cases returns the number of test cases executed

SYNOPSIS

function count = count_test_cases(self)

DESCRIPTION 

test_suite/count_test_cases returns the number of test cases executed 
by run.

  Example
  =======
         suite = mlunit_all_tests;
         count_test_cases(test);     % Returns 26 with version 1.4.
